# Toss Simulator 🪙

A **C++ console game** that simulates coin tosses for multiple players. Designed for fun, practice with **C++17**, and competitive tossing scenarios!

---

## 🕹 Game Overview

- Flip a coin virtually with 1 or more participants.
- Choose game modes:
  - **Regular:** Single toss for everyone.
  - **Ranked:** Elimination style with rankings.
  - **Best of 3:** Decide the winner over 3 tosses between 2 players.
- Real-time coin flipping animation for visual flair.
- Displays the winner clearly with colored output.

---

## 🛠 How to Compile & Run

### Compile with **C++17** (recommended):

```bash
g++ -std=c++17 Toss_Simulator.cpp -o Toss_Simulator
